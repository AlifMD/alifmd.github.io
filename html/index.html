Semua Tentang Saya

Perkenalkan nama lengkap saya Alif Muhammad Darussalam Al Khairin biasa dipanggil Alif atau Alifmd (saat sd). Saya adalah putra tunggal. Ayah saya berkerja bekerja di IPB tetapi bukan sebagai dosen, beliau bekerja sebagai PNS. Ibu saya berkerja sebagai ibu rumah tangga.

Saya pertama kali masuk sekolah di Taman Kanak-kanak Aqila,berada di Komplek IPB Alam Sinarsari di Dramaga. Kemudian saya melanjutkan ke SDIT Al-Yasmin dan melanjutkannya lagi ke tempat yang sama yaitu SMPIT Al-Yasmin. Dan sekarang saya berada di SMK-SMAK Bogor SMK-SMAK Bogor, di SMK ini saya mengikuti ekstrakulikuler Voli. Saya mempunyai niat untuk melanjutkan bekerja dari SMK-SMAK Bogor untuk bekerja.
